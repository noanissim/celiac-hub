## 🌟 Project Overview

**CeliacHub** is a comprehensive portal designed to support the gluten-free community in Israel and abroad. It features:

* 
**Dynamic Restaurant Directory**: Filterable list of restaurants with GF options across Israel, including specific focus on Netanya and Tel Aviv.


* 
**Interactive Maps**: Interactive pins for coffee carts with gluten-free options based on CoffeeTrail data.


* **Research & News**: A dynamic feed of the latest celiac research fetched via an automated pipeline.
* **Community Features**: User authentication via Google, personalized "Favorites" lists, and community comments.

## 🛠️ Tech Stack

* **Frontend**: React, Tailwind CSS, Lucide Icons.
* **Backend & Database**: Supabase (PostgreSQL).
* **Logic**: Lovable (AI-driven development) and Edge Functions.
* **Data Pipeline**: Python-based ingestion for real-time research updates.
